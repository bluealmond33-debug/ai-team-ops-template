# KDA4 공유 메시지 초안

KDA4 분들께 공유드려요.

AI 에이전트를 그냥 챗봇처럼 쓰는 게 아니라, 작은 팀처럼 운영하기 위한 GitHub 템플릿을 만들었습니다.

- GitHub Issue로 작업 정리
- 에이전트 역할 나누기
- handoff 양식으로 작업 위임
- Obsidian식 폴더에 결정/회고 저장
- PR/리뷰/브리프로 검증

완성품이라기보다 fork해서 자기 프로젝트에 맞게 고쳐 쓰는 "슈퍼 샘플"에 가깝습니다. 토큰이나 개인 데이터 없이 demo mode로 구조만 먼저 볼 수 있게 만들었습니다.

편하게 베끼고, 고치고, 더 좋은 방식 있으면 알려주세요.
