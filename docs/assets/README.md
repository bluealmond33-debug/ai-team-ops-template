# README 이미지 자산

이 폴더는 GitHub README와 KDA4 공유용 public-safe SVG 이미지를 담습니다.

## 포함 파일

- `ai-team-flow.svg`: AI 팀 운영 전체 흐름
- `github-fork-guide.svg`: GitHub Fork 버튼 안내
- `handoff-example.svg`: handoff 문서 예시
- `issue-example.svg`: GitHub Issue식 작업 분해 예시
- `obsidian-vault-example.svg`: Obsidian식 기록 폴더 예시

## 원칙

- 실제 채팅방, 실제 프로젝트 내부 정보, 실제 토큰/ID를 사용하지 않습니다.
- 모두 설명용 가짜 UI입니다.
- SVG라서 GitHub README에서 바로 보이고, 확대해도 깨지지 않습니다.
