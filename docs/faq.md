# FAQ

## 실제 API 키가 필요한가요?

아니요. 처음에는 demo mode로 구조만 확인하면 됩니다.

## Telegram이나 Discord를 꼭 써야 하나요?

아니요. 메시지 채널은 Slack, Teams, 카카오워크 등으로 바꿀 수 있습니다.

## 개인 노트를 넣어도 되나요?

로컬에서는 가능하지만 public repo에는 넣지 마세요. 예시에는 synthetic data만 사용하세요.

## 이건 자동화 도구인가요, 운영 방식인가요?

둘 다입니다. 핵심은 도구보다 작업을 나누고, 검증하고, 기록하는 운영 방식입니다.
