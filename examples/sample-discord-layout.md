# Sample Discord/Telegram Layout

## Minimal version

- `start`: how to use the workspace
- `requests`: user asks for work
- `handoffs`: agents receive and return work
- `reviews`: reviewer feedback
- `briefs`: daily/weekly summaries

## Naming tip

Use simple names first. Add more channels only when the team actually needs them.

## Privacy tip

Do not paste private logs, personal notes, real chat IDs, or tokens into public examples.
